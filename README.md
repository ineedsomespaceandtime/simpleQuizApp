
# Simple Quiz App

A vibe-coded quiz app that lets you put your questions and answers, with the option to put text based questions and image based questions. It also allows you to put identification type of questions. 
## Features

- Google Gemini for the Hints
- Image support for the questions
- Multiple Choice questions
- Identification Questions
## Installation

Install my-project with npm

```bash
  npm install simpleQuizApp
  cd simpleQuizApp
```
    
## Tech Stack

**Client:** HTML, CSS, Javascript






## Run Locally

Clone the project

```bash
  git clone https://github.com/ineedsomespaceandtime/simpleQuizApp.git
```

Go to the project directory

```bash
  cd simpleQuizApp
```

Go to Google AI studio and create your API key

```bash
  Go to the dashboard and create your API key
```

Change your API key in script.js

```bash
  const GEMINI_API_KEY='PUT-YOUR-API-KEY-HERE';
```
